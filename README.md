# SOLPLAY FARM 🌱

Play • Plant • Profit (Solana Game)

## Stack
- Frontend: Next.js
- Backend: Node.js (Express)
- Blockchain: Solana (Web3.js)

## Setup

### 1. Backend
cd backend  
npm install  
node server.js  

### 2. Frontend
cd frontend  
npm install  
npm run dev  

## Notes
- Replace TREASURY wallet before production
- This version uses backend validation (Phase 1)
- Phase 2 = fully on-chain (Anchor)
